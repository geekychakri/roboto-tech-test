# Tech Test

## Demo

[roboto-tech-test-chakradhar.vercel.app](https://roboto-tech-test-chakradhar.vercel.app)

## Run Locally

Clone the project

```bash
  git clone https://github.com/geekychakri/roboto-tech-test.git
```

Go to the project directory

```bash
  cd roboto-tech-test
```

Install dependencies

```bash
  pnpm install
```

Start the project

```bash
  pnpm run dev
```
